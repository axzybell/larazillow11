## Features

## Tech Stack
- Laravel 12
- Vue.js 3
- Inertia.js
- MySQL (Docker)
- Mailhog (Docker)

## Getting Started
1. Clone the repo
2. Run `composer install` and `npm install`
3. Set up `.env` and run migrations
4. Serve the app with `php artisan serve`
